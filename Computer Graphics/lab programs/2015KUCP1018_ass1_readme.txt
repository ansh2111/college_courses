Compile and Execute the file 2015KUCP1018_ass1_cohen.c. make sure you have graphics.h installed on your system.
gcc 2015KUCP1018_ass1_cohen.c -lgraph
./a.out

Input:xmin, xmax,ymin,ymax coordinates
      x1,x2,y1,y2
Output: line rejected or Accepted from (x1,y1) to (x2,y2).

Example:
Input:
Enter the coordinates of window
xmin, xmax, ymin, ymax:0 5 0 5
Enter cordinates of the line
x1, y1, x2, y2:-1 0 4 6
Press Ctrl+C to Continue or Quit

Output:
Line Accepted from (0.00,1.20) to (3.17,5.00).
     
     
Note: Press Ctrl+C to exit the initial graphic window. and then another graphics window will appear showing resultant graph. To quit program press Ctrl+c.
