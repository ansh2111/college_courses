There is a file for DES encryption. Make sure there is a file 2015KUCP1018_ass8_sbox.txt exists to take sbox values as input.
Execute It:-

gcc 2015KUCP1018_ass8_des.c -lm
Input: 	<PLAIN-TEXT(64)>

Input: <KEY(64)>

Output: <CIPHER-TEXT(64) ASCII VALUES>

Example:
Enter Plain Text(8-bytes):PQRSTUVW

Enter Key(8-bytes):hello987

ASCII Values of Cipher Text:
122 57 222 91 229 54 92 54 
Cipher Text: z9 [ 
6
\6
(Some values belong to Extended ASCII set so they can't be displayed on screen. Here 3rd and 5th characters.)
