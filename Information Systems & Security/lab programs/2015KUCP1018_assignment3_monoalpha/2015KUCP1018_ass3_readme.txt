There are 2 files for encryption and decryption.

Compile and Execute Them:-

gcc 2015KUCP1018_ass3_monoalphaen.c
./a.out

Input: <PLAIN-TEXT>

Output: <CIPHER-TEXT>

gcc 2015KUCP1018_ass3_monoalphade.c
./a.out

Input: <ENCRYPTED-TEXT>

Output: <DECRYPTED-TEXT>
