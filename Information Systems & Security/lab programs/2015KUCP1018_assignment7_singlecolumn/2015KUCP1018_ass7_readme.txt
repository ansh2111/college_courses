There is a single file for encryption and decryption.

This a python3 file. So, make sure you have python 3.5 installed on system and random module also installed since this program uses Random library.
Execute It:-

python 2015KUCP1018_ass7_singlecolumn.py
Input: 	<PLAIN-TEXT>

	<COLUMN-COUNT>
Output: <CIPHER-TEXT>

Output: <DECRYPTED-TEXT>

Example:
Enter Plain Text:jaimaheshmati
Column Count: 5

Cipher Text: mhxjhaamxaetisi

Decipher Text: jaimaheshmati
