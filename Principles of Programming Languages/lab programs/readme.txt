Assignments List
1. Write a program in LISP for Fibonacci series.
2. Program in LISP for intersection, union and difference of two lists.
3. Write a program in LISP to perform swapping on array in following manner: 
   3.1. swap first and second element , 3rd and 4th element and so on. 
   3.2. swap first and last element, second and second last element and so on.
4. Write a program in LISP to perform following sorting on array: 
   4.1. Bubble sort 
   4.2. Insertion Sort
5. Write a program in LISP to perform find and replace activity, where you take a paragraph as input and find any word(input take by user) in the paragraph if it is present then replace it by any other word.
